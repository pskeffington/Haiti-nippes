# Project status

**Repository status:** validation-gated maternal-health access framework and public-health policy analysis scaffold.

Haiti Nippes is an active public-health repository focused on maternal-health access, rural referral constraints, emergency obstetric care readiness, WASH resilience, and commune-level prioritization in Nippes Department, Haiti.

## Current output

The repository now contains a structured abstract, an abstract ML-readiness checklist, a scanner-friendly abstract evaluation packet, a maternal-health framework, a maternal-access prioritization index, a commune-level input template, and source-registry entries for national and departmental maternal-health source leads.

## STEM CV Curator surface

This project should be classified under public health, infrastructure, and environmental evidence. It is a validation-gated scaffold because the object model and abstract are ready for review, but department-level and commune-level quantitative claims must remain source-tagged before being used in final public outputs.

## Review-ready abstract claim

The project develops a source-tagged maternal-health access framework for Nippes Department, Haiti, to support commune-level prioritization of rural maternal referral constraints and emergency obstetric care readiness. It separates national Haiti maternal-health indicators from Nippes-specific analytic inputs and defines the maternal-access index as a prioritization tool rather than an official maternal mortality estimate.

## Evidence boundary

- National maternal-health claims may use PAHO, WHO, World Bank, UNFPA, and MSPP source leads when fully reconciled.
- Nippes-specific claims require departmental or commune-level source verification.
- Placeholder CSV rows are templates and must not be interpreted as observed data.
- The maternal-access score is exploratory and should not be described as causal, predictive, or official.
- No field verification is implied unless independently documented.

## Current maturity

Validation-gated source and object-model scaffold. Ready for abstract scanning and CV curation; not yet ready for final empirical claims.
